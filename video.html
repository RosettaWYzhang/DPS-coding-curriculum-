Introduction to knight's Tour Problem

Knight’s tour is a sequence of movement of a chess piece, knight. The knight must visit all of the squares on a chessboard for a successful tour, and the same square can only be visited once. A knight can make a movement such as; move two squares horizontally, then one square vertically, OR two squares vertically then horizontally once. So its like a shape of an L, or a rotated L, or a reversed L, or both. The size of a chessboard does not matter in knight’s tour. However with some chessboard sizes, you are not able to make a complete tour. If a tour is closed, that means from the very last square visited, you can go back to the starting square. If a tour is opened, it means you cannot go back to the starting square from the last square.

Knight's Tour Computation

We will be trying to solve knight’s tour by programming, but first we can compute knight’s tour on our own first, to familiarise ourselves to the problem. Playing the knight’s tour game is a good way familiarise with the problem.You can construct a tree to represent the movements of a knight on a chessboard. The starting square is the root of the tree. You can add nodes to the tree, by adding the squares you have visited. Record the other square you did not visit as well. Come back to the square you did not visit when you reach a dead end. Continue to construct the tree from the square(node) you have visited.

Backtracking 1


//pseudocode
If all squares are visited
  print the solution
Else
  a) Add one of the next moves to solution vector and recursively
  check if this move leads to a solution. (A Knight can make maximum
  eight moves. We choose one of the 8 moves in this step).
  b) If the move chosen in the above step doesn't lead to a solution
  then remove this move from the solution vector and try other
  alternative moves.
  c) If none of the alternatives work then return false (Returning false
  will remove the previously added item in recursion and if false is
  returned by the initial call of recursion then "no solution exists")

#include < stdio.h>
#include < stdbool.h>
#define N 8

int solveKTUtil(int x, int y, int movei, int sol[N][N],
 int xMove[],  int yMove[]);

/* A utility function to check if i,j are valid indexes
   for N*N chessboard */
bool isSafe(int x, int y, int sol[N][N])
{
    return ( x >= 0 && x < N && y >= 0 &&
             y < N && sol[x][y] == -1);
}

/* A utility function to print solution matrix sol[N][N] */
void printSolution(int sol[N][N])
{
    for (int x = 0; x < N; x++)
    {
        for (int y = 0; y < N; y++)
            printf(" %2d ", sol[x][y]);
        printf("\n");
    }
}
/* A recursive utility function to solve Knight Tour
   problem */
int solveKTUtil(int x, int y, int movei, int sol[N][N],
                int xMove[N], int yMove[N])
{
   int k, next_x, next_y;
   if (movei == N*N)
       return true;

   /* Try all next moves from the current coordinate x, y */
   for (k = 0; k < 8; k++)
   {
       next_x = x + xMove[k];
       next_y = y + yMove[k];
       if (isSafe(next_x, next_y, sol))
       {
         sol[next_x][next_y] = movei;
         if (solveKTUtil(next_x, next_y, movei+1, sol,
                         xMove, yMove) == true)
             return true;
         else
             sol[next_x][next_y] = -1;// backtracking
       }


   return false;
}

bool solveKT()
{
    int sol[N][N];

    /* Initialization of solution matrix */
    for (int x = 0; x < N; x++)
        for (int y = 0; y < N; y++)
            sol[x][y] = -1;

    /* xMove[] and yMove[] define next move of Knight.
       xMove[] is for next value of x coordinate
       yMove[] is for next value of y coordinate */
    int xMove[8] = {  2, 1, -1, -2, -2, -1,  1,  2 };
    int yMove[8] = {  1, 2,  2,  1, -1, -2, -2, -1 };

    // Since the Knight is initially at the first block
    sol[0][0]  = 0;

    /* Start from 0,0 and explore all tours using
       solveKTUtil() */
    if (solveKTUtil(0, 0, 1, sol, xMove, yMove) == false)

        printSolution(sol);

   tour is possible, otherwise return true and prints the

   this function prints one of the feasible solutions.  */
bool solveKT()
{
    int sol[N][N];

    int yMove[8] = {  1, 2,  2,  1, -1, -2, -2, -1 };

    // Since the Knight is initially at the first block
    sol[0][0]  = 0;

    /* Start from 0,0 and explore all tours using
       solveKTUtil() */

    {
        printf("Solution does not exist");
        return false;
    }
    else
        printSolution(sol);

    return true;
}
