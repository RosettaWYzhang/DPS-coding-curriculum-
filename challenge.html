Find reverse

Write a Program which reverses and input. eg you enter 12345 and the program returns 54321.

Solution


#include <stdio.h>
int main()
{
   printf("Enter what you want reversed(max 100 characters)\n");
   char reverse [100];
   // to save what we want reversed
   int i,j;
   for(i = 0; reverse[i-1]!='\n';i++){
      // cycle until the previous item was the enter key '\n' mean enter key
      scanf("%c",&reverse[i]);
      // we cycle through and save each chracter in the next array space
   }
   printf("i is %i",i);
   for(j=i-1;j>=0;j--){
      // we know how long it is from int i; so we just loop backwards through the array
      printf("%c",reverse[j]);
   }
   printf("\n");
  return 0;
}

Find product

Write a program which take two integer inputs, e.g. 4 and 10 , and calculate the product of all the numbers between these two integers. 4*5*6*7*8*9*10.

Solution

   int first, second;

   scanf("%i",&first);
   printf("please enter second digit: ");
   scanf("%i",&second);

   int higher = second;
   // its useful to know which is the larger of the two
   int lower = first;
   if(first>lower){
      first = higher;
      // set the highest one to highest
      second = lower;
   }
   int product = 1;
   for(;lower<=higher;lower++){
     // loops from lowest to highest going up by one every time, multiplyinh the old product by the new number
      product *=lower;

      } else {
         printf("%i*",lower);
      }
   }
   printf("%i\n",product);

Tutorial

For loop:For loops as you learnt have the form like this; for(initialise; condition; increment) and you can also have for loops within another for loop which is a nested for loop.
An example of nested for loop: 


for(i = 0; i < 5; i++) {
  for(j = 0; j < 5; j++) {
    printf(“*”);
  }
    printf(“\n”);
}
What does the result of this part of the code look like? It looks like this:
*****
*****
*****
*****
*****
If we look at this in steps:
  • the first for loop (loop with i)starts, then the program enters the second loop(loop with j).
  • int the second loop, the printf is applied, * is printed.
  • The second loop repeats the print for 5 times. (j = 0,1,2,3,4 : 5 counts)
  • After the loop has ended, the printf(“\n”); is printed, which just means move to new line.
  • The code reaches end of the first for loop, so that is one count for the first loop, value of i increases and goes back to the beginning of the loop, meaning, the second loop starts again.
So, in simple words, this part of the code repeats to print “*” 5 times, for 5 times.

Switch statement Switch statements are similar to if statements. An example of switch statement: 

switch(number) {
   case 1:
      printf(“1”);
      break;
   case 2:
      printf(“2”);
      beak;
  case 3:
      printf(“3”);
      break;
}
The “number” may be assigned with a type int, by either the code or input such as by scanf.

What the switch statement does it it looks at what is inside the bracket; switch(here). Inside the parenthesis tells the code what to do for whatever the number is.

In this case the number may be either 1,2, or 3, and the code will print one of the numbers depending on the value of “number”.

If the number is 1, it prints 1, if number is 2, it prints 2.

Some difference with switch/case with if/else may be that switch/case can only have discrete condition, and if/else can have a range of values. However in some cases it may be easier to write switch/case, rather than many if/else statements.

Tower of Hanoi

The Tower of Hanoi is a mathematical puzzle. It consists of three rods, and a number of disks in a neat stack in ascending order of size on one rod, the smallest at the top, thus making a conical shape.
The objective of the puzzle is to move the entire stack to another rod obeying:

  1. only one disk can be moved at a time.
  2. each move consists of taking the upper disk from one of the stacks and placing it.
  3. no disk may be placed on top of a smaller disk.

In order to solve tower of hanoi more simply, it’s better for us to do the following procedures:
  • label the pegs as A, B, C
  • let n be the total number of disks
  • number the disks from topmost as 1 to the bottom-most as n
Assuming all n disks are distributed among three pegs, we can move these disks from a source peg which to a target peg using a spare peg using the following rules recursively: move n-1 discs from the source to the spare peg, by the same general solving procedure. This step should eave the disc n as a top disk on the source peg. move the disc n from the source to the target peg move the n-1 disks that we’ve just placed on the spare peg to the target peg by the same general saving procedure. if the number of disk is 0, then do nothing.

The above is a recursive algorithm to the tower of hanoi and is actually pretty like pseudocode for this problem. So if we want to make a program to solve this puzzle, we just need translate those pseudocode into actual code. And the following is the actual pseudocode:


// pseudocode
A = [5,4,3,2,1]
B = [ ]
C = [ ]

def move(n, source, target, auxiliary)
	if n > 0
		//move n-1 disks from source to spare peg
		move(n-1, source, auxiliary, target)

		//move the nth disk from source to target
		moveOne(source, target)

		//move the n-1 disks on the spare peg to target peg
		move(n-1, auxiliary, target, source)

void swap(int n, char start, char final)
{
    switch(start)
    {
        case 'A':
            A[n] = '\0';
            break;
        case 'B':
            B[n] = '\0';
            break;
        case 'C':
            C[n] = '\0';
            break;
        default:
            break;
    }

    switch (final)
    {
        case 'A':
            A[n] = n;
            break;
        case 'B':
            B[n] = n;
            break;
        case 'C':
            C[n] = n;
        default:
            break;
    }
}


void transfer(int num,char start,char trans,char final)
{
    if(num == 1)
    {
        printf("\nMove disk 1 from peg %c to peg %c \n", start, final);
        swap(1,start,final);
        display();
    }
    else
    {
        transfer(num - 1, start, final, trans);
        printf("Move disk %d from peg %c to peg %c \n", num, start, final);
        swap(num,start,final);
        display();
        transfer(num - 1, trans, start, final);
    }
}

Quick Sort

Sort the array {41,6,3,2,21,9,7} using quick sort according to the description and pseudocode given

Tutorial

Every programmer should be familiar with sorting, an important area in algorithms. Here, we introduce a highly efficient sorting algorithm called quick sort invented by Tony Hoare. The average-case time complexity is O(nlgn), which is better than the average-case time complexity of bubble sort and insertion sort.

Given an array, an element known as the pivot is chosen. The rest of the elements are divided into two subsets according to whether they are bigger or smaller than the pivot element. Using recursion, the above procedure is applied to smaller subsets until each subset contains only 1 element. The picture below shows how quick sort works.

Pseudocode


  function quicksort('array')
    if length('array') ≤ 1
       return 'array'
       // an array of zero or one elements is already sorted
    select and remove a pivot value 'pivot' from 'array'
    create empty lists 'less' and 'greater'
    for each 'x' in 'array'
       if 'x' ≤ 'pivot' then append 'x' to 'less'
       else append 'x' to 'greater'
    return concatenate(quicksort('less'), 'pivot', quicksort('greater'))

Solution


//reference: https://www.tutorialspoint.com/data_structures_algorithms/quick_sort_program_in_c.htm
#include <stdio.h>
#include <stdbool.h>
#define MAX 7

int intArray[MAX] = {41,6,3,2,21,9,7};

void printline(int count) {
   int i;

   for(i = 0;i <count-1;i++) {
      printf("=");
   }

   printf("=\n");
}

void display() {
   int i;
   printf("[");

   // navigate through all items
   for(i = 0;i<MAX;i++) {
      printf("%d ",intArray[i]);
   }

   printf("]\n");
}

void swap(int num1, int num2) {
   int temp = intArray[num1];
   intArray[num1] = intArray[num2];
   intArray[num2] = temp;
}

int partition(int left, int right, int pivot) {
   int leftPointer = left -1;
   int rightPointer = right;

   while(true) {
      while(intArray[++leftPointer] < pivot) {
         //do nothing
      }

      while(rightPointer > 0 && intArray[--rightPointer] > pivot) {
         //do nothing
      }

      if(leftPointer >= rightPointer) {
         break;
      } else {
         printf(" item swapped :%d,%d\n", intArray[leftPointer],intArray[rightPointer]);
         swap(leftPointer,rightPointer);
      }
   }

   printf(" pivot swapped :%d,%d\n", intArray[leftPointer],intArray[right]);
   swap(leftPointer,right);
   printf("Updated Array: ");
   display();
   return leftPointer;
}

void quickSort(int left, int right) {
   if(right-left <= 0) {
      return;
   } else {
      int pivot = intArray[right];
      int partitionPoint = partition(left, right, pivot);
      quickSort(left,partitionPoint-1);
      quickSort(partitionPoint+1,right);
   }
}

int main(void) {
   printf("Input Array: ");
   display();
   printline(50);
   quickSort(0,MAX-1);
   printf("Output Array: ");
   display();
   printline(50);
}
